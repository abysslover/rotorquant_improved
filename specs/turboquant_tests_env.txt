# TurboQuant Test Environment

## Conda Environment

**Environment Name**: `py310`  
**Python Version**: `3.10.x`

### Environment Creation

```bash
# Create environment(if not exists)
conda create -n py310 python=3.10 -y
conda activate py310

# Install core dependencies
pip install torch>=2.0.0 triton>=3.0 numpy scipy

# Install validation dependencies (optional)
pip install transformers datasets bitsandbytes

# Install package in editable mode
pip install -e .
```

### Environment Verification

```bash
# Verify Python version
python --version  # Should be 3.10.x

# Verify PyTorch
python -c "import torch; print(f'PyTorch {torch.__version__}')"

# Verify CUDA (if available)
python -c "import torch; print(f'CUDA available: {torch.cuda.is_available()}')"
```
