## 2. Detailed Test Specifications

### 2.1 T1: MSE Distortion (Core)

**Purpose**: Verify quantization matches theoretical bounds

**Config**:
- Dimensions: d = [64, 128, 256]
- Bit widths: bits = [1, 2, 3, 4]
- Vectors: n = 2000 unit vectors
- Device: cpu

**Procedure**:
```python
for d in [64, 128, 256]:
    for bits in [1, 2, 3, 4]:
        quantizer = TurboQuantMSE(d, bits, seed=42, device="cpu")
        x = torch.randn(2000, d)
        x = x / x.norm(dim=-1, keepdim=True)
        x_hat, _ = quantizer(x)
        mse = ((x - x_hat) ** 2).mean()
        
        theory = (sqrt(3) * pi / 2) * (1 / (4 ** bits))
        ratio = mse / theory
        
        assert ratio < 1.5, f"MSE ratio {ratio} exceeds 1.5× theory"
```

**Expected Results**:

| bits | Theory | Expected MSE | Max Allowed |
|------|--------|--------------|-------------|
| 1 | 0.680 | ~0.5 | 1.02 |
| 2 | 0.170 | ~0.12 | 0.255 |
| 3 | 0.0425 | ~0.034 | 0.064 |
| 4 | 0.0106 | ~0.009 | 0.016 |

### 2.2 T2: Inner Product Unbiasedness (QJL)

**Purpose**: Verify QJL provides unbiased inner product estimates

**Config**:
- d = 128
- n_trials = 2000
- bits = [2, 3, 4]

**Procedure**:
```python
for bits in [2, 3, 4]:
    quantizer = TurboQuantProd(d, bits, seed=42)
    
    x = torch.randn(n_trials, d) / norm
    y = torch.randn(n_trials, d) / norm
    
    true_ip = (x * y).sum(dim=-1)
    comp = quantizer.quantize(x)
    est_ip = quantizer.inner_product(y, comp)
    
    bias = (est_ip - true_ip).mean()
    rmse = ((est_ip - true_ip) ** 2).mean().sqrt()
    corr = corrcoef(true_ip, est_ip)[0, 1]
    
    assert abs(bias) < 0.01, f"Bias {bias} too large"
    assert corr > 0.95, f"Correlation {corr} too low"
```

**Expected Results**:

| bits | Expected Bias | Expected RMSE | Expected Corr |
|------|---------------|---------------|---------------|
| 2 | ~0.001 | ~0.04 | > 0.85 |
| 3 | ~0.0006 | ~0.04 | > 0.87 |
| 4 | ~0.0007 | ~0.02 | > 0.96 |

### 2.3 T8: Attention Score Cosine Similarity

**Purpose**: Verify compressed KV cache produces similar attention scores

**Config**:
- Model: Qwen2.5-3B (2 KV heads)
- seq_len = [1024, 4096, 8192]
- bits = [2, 3, 4]

**Procedure**:
```python
# Load model and get KV cache
model = AutoModelForCausalLM.from_pretrained("Qwen/Qwen2.5-3B-Instruct")
cache = model(inputs).past_key_values

for bits in [2, 3, 4]:
    for layer_idx in range(n_layers):
        keys = cache[layer_idx].keys  # (1, H, S, D)
        
        key_comp = TurboQuantCompressorV2(D, bits, seed=layer_idx)
        compressed_k = key_comp.compress(keys)
        
        query = keys[:, :, -1:, :]  # Last token query
        real_scores = query @ keys.transpose(-2, -1)
        tq_scores = key_comp.asymmetric_attention_scores(query, compressed_k)
        
        # Cosine similarity of score vectors
        cos_sim = cosine_similarity(real_scores, tq_scores)
        
        assert cos_sim > 0.99, f"Cosine sim {cos_sim} too low"
```

**Pass Criteria**:
- Cosine similarity > 0.99
- Top-1 match rate > 95%
- Top-5 match rate > 99%

### 2.4 T11: Needle-in-Haystack

**Purpose**: Verify compressed cache can retrieve specific tokens

**Config**:
- seq_len = [512, 2048, 8192]
- needle_pos = seq_len // 3
- bits = [2, 3, 4]

**Procedure**:
```python
keys = torch.randn(seq_len, d) / norm
needle = keys[needle_pos].clone()
query = needle.unsqueeze(0)

quantizer = TurboQuantProd(d, bits)
compressed = quantizer.quantize(keys)
ips = quantizer.inner_product(query.expand(seq_len, -1), compressed)

top_idx = ips.argmax().item()
found = (top_idx == needle_pos)
top5 = ips.topk(5).indices.tolist()
in_top5 = needle_pos in top5

assert found or in_top5, f"Needle not found: top={top_idx}"
```

**Pass Criteria**:
- EXACT match for all configs
- Top-5 coverage: 100%

### 2.7 T21-23: Cross-Engine Consistency

**Purpose**: Verify all engines produce identical results

**Config**:
- d = 64, bits = 3
- Seeds: 42
- Backends: [python, numpy, cuda, triton]

**Procedure**:
```python
engines = ["python", "numpy"]
if cuda_available: engines.extend(["cuda", "triton"])

mse_results = {}
memory_results = {}

for backend in engines:
    quantizer = TurboQuantFactory.create_quantizer(
        method="turboquant", backend=backend, d=d, bits=bits, seed=42
    )
    
    x = torch.randn(100, d)
    x_hat, _ = quantizer(x)
    mse = ((x - x_hat) ** 2).mean().item()
    mse_results[backend] = mse
    
    cache = TurboQuantFactory.create_kvcache(
        method="turboquant", backend=backend,
        d_key=d, d_value=d, key_bits=4, value_bits=2, seed=42
    )
    cache.append(torch.randn(10, d), torch.randn(10, d))
    memory_results[backend] = cache.memory_usage_bits()["total_bits"]

# Verify consistency
baseline_mse = mse_results["python"]
baseline_memory = memory_results["python"]

for backend, mse in mse_results.items():
    if backend != "python":
        diff = abs(mse - baseline_mse) / baseline_mse
        assert diff < 0.01, f"MSE diff {diff:.3f} exceeds 1%"

for backend, memory in memory_results.items():
    if backend != "python":
        assert memory == baseline_memory, f"Memory mismatch: {memory} vs {baseline_memory}"
```

**Pass Criteria**:
- MSE difference < 1% across all engines
- Memory calculation exact match

## 4. QJL-Specific Tests (Research)

### 4.1 QJL Variance Analysis

**Purpose**: Investigate why QJL + softmax fails

**Procedure**:
```python
# Compare variance of attention scores
x = torch.randn(1000, d) / norm
y = torch.randn(1000, d) / norm

# MSE-only
quant_mse = TurboQuantMSE(d, bits=3)
x_hat_mse, _ = quant_mse(x)
scores_mse = x_hat_mse @ y.T
var_mse = scores_mse.var()

# QJL
quant_qjl = TurboQuantProd(d, bits=3)
comp_qjl = quant_qjl.quantize(x)
scores_qjl = quant_qjl.inner_product(y, comp_qjl).unsqueeze(-1) @ y.T
var_qjl = scores_qjl.var()

# Expected: var_qjl >> var_mse
print(f"MSE variance: {var_mse:.6f}")
print(f"QJL variance: {var_qjl:.6f}")
print(f"Ratio: {var_qjl / var_mse:.2f}×")
```

**Hypothesis**: QJL variance should be 10-100× higher than MSE-only

### 4.2 QJL Implementation Verification

**Purpose**: Verify QJL implementation matches paper

**Checklist**:
- [ ] S matrix is random orthogonal (or as specified)
- [ ] qjl_signs are ±1 (not 0/1 or other values)
- [ ] Residual norm computed correctly
- [ ] Inner product formula: `<y, x_hat> + scale * <y_proj, signs>`
- [ ] Scale factor = sqrt(π/2) / qjl_dim

**Reference**: QJL paper (arXiv:2406.03482) Section 3

## 4.5 T15: Asymmetric Bits Allocation

**Purpose**: Verify exact bit allocation for keys vs values

**Config**:
- d_key = 128, d_value = 128
- seq_len = 100
- key_bits = 4, value_bits = 2

**Procedure**:
```python
cache = TurboQuantFactory.create_kvcache(
    method="turboquant", backend="python",
    d_key=128, d_value=128,
    bits=3, key_bits=4, value_bits=2, seed=42
)

keys = torch.randn(100, 128)
values = torch.randn(100, 128)
cache.append(keys, values)

usage = cache.memory_usage_bits()

# Verify exact bit calculation
expected_key_indices = seq_len * d_key  # 100 * 128 = 12800
expected_value_indices = seq_len * d_value  # 100 * 128 = 12800
expected_bits = expected_key_indices * 4 + expected_value_indices * 2
# = 12800 * 4 + 12800 * 2 = 51200 + 25600 = 76800

assert usage["total_bits"] == expected_bits, \
    f"Expected {expected_bits} bits, got {usage['total_bits']}"
```

**Pass Criteria**:
- Total bits exactly matches formula: key_indices × key_bits + value_indices × value_bits
- Compression ratio correct
